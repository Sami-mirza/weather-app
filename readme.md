# Weather App 🌤️

A Python app that fetches real-time weather data for any city using the OpenWeatherMap API.

## Features
- Get current temperature, humidity, wind speed and weather condition
- Error handling for invalid city names

## How to run
1. Clone this repo
2. Install dependencies: `pip install requests python-dotenv`
3. Create a `.env` file and add your API key: `API_KEY=your_key_here`
4. Run: `python main.py`
